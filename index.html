🌱 חזון

כנר לרגלי דבריך: ליצור כלים פתוחים שמחברים קוד וחכמה רוחנית. אור לנתיבתי: להנגיש תפילות, תבניות ושירותים דיגיטליים שמשרתים קהילה.

אפשרויות פרקטיות: דפי A4 נגישים, תבניות להדפסה, מאגרי טקסטים רוחניים ו-APIs לשילוב בבוטים ויישומים.

🛠 כישורים וטכנולוגיות

  • שפות: Python, JavaScript, TypeScript
  • Frontend: React, Next.js, CSS מודרנית, עיצוב נגיש
  • Backend: Node.js, FastAPI, שילובי API ל-AI
  • DevOps: Docker, GitHub Actions, CI/CD

🚀 פרויקטים מרכזיים

  1. agi-rabbi — מאגר שאלות ותשובות רוחניות משולב AI.
  2. accessible-prayers — דפי תפילה בעיצוב A4 נגיש להדפסה.
  3. spirit-tools — ספריות תבניות ו-CLI ליצירת טקסטים רוחניים.
